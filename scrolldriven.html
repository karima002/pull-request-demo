Scroll-Driven Animation on images (CSS Only)

To create a smooth scroll-driven animation on the image, I used CSS features like animation-timeline and view(), which allow animations to be triggered based on the scroll position. Animation-timeline triggers animation when the element enters the viewport. Here is the CSS used specifically for the images:

                
                    @media (prefers-reduced-motion: no-preference) {
                        .image-container {
                            position: absolute;
                            top: 25%;
                            left: 25%;
                            width: 50vw;
                            height: 50vh;
                            animation: fade-in linear;
                            animation-timeline: view();   
                            animation-range: entry; 
                        }

                        @keyframes fade-in {
                            from {
                                scale: 0.7;
                                opacity: 0;
                            }

                            to {
                                scale: 1;
                                opacity: 1;
                            }
                        }
                    }
